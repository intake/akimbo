Installation
============

Requirements
~~~~~~~~~~~~

To install ``akimbo`` you will need ``awkward`` and
one of the backend libraries: ``pandas``, ``dask`` or ``polars``.


From PyPI
~~~~~~~~~

.. code-block:: none

   $ pip install akimbo

From conda-forge
~~~~~~~~~~~~~~~~

.. code-block:: none

   $ conda install akimbo -c conda-forge
